## Stick Notes Chrome Extension

### Local development
```bash
npm install
npm run dev
//browser should open to localhost:4040/public
```

### Testing on Chrome
```bash
npm install
npm run build
```
1. Go to chrome://extensions
1. Enable developer mode (_top right corner_)
1. Load upacked > navigate to `public/` directory
1. Open new tab ...
